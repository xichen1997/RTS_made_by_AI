Procedural RTS Prototype

This lightweight prototype ports the procedural terrain explorer to the web. Generate new worlds, spawn explorer units and command them around the map directly in your browser.

How to play

  1. Press Spawn unit and click the map to place explorers.
  2. Click a unit to select it. A highlighted outline indicates selection.
  3. Click anywhere else on the map to set the selected unit's destination.
  4. Use New world to regenerate the map. Provide a seed for reproducible results.

Debug helpers

The panel below mirrors the in-memory game state so it is easier to debug and extend the experience. Use it to verify positions, targets and selection state.
